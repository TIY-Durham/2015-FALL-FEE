# How do I put the values `1`, `2`, and `3` into an `Array` named `numbers`?

* `numbers = [ 1, 2, 3 ]`
* `numbers[0] = 1; numbers[1] = 2; numbers[2] = 3;`
* `numbers.push(1, 2, 3)`

# How do I create an _empty_ dictionary named `people`?

* `people = { }`

* `people.kelly = "Associate Instructor";`
